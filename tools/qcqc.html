QCQC

大層な名前がついておりますが、そんなに大したコードではありません。
要は「クリップボードにある文字列」から、指定した文字で始まるものを抜き出して
クリップボードに再貼付けするだけのスクリプトです。
使用前にPythonをインストールし、 
pip install pyperclip
を実行しておいて下さい。 
import pyperclip

print("クリップボードから指定文字列を含むやつを抜き出すくん\n")
outList = []
query = input("指定する文字列を入力せよ>")
urlList = pyperclip.paste().split()
for url in urlList:
    if url.startswith(query):
        outList.append(url)
pyperclip.copy("\n".join(outList))

活用法

私は気になった動画をYoutubeで次々と開いちゃうスタイルなので、タブが溜まりがちです。
そんな時、上記のスクリプトを パスが通ったフォルダにqcqc.pyとして保存した後、 
python qcqc.py
としたqcqc.batを同じフォルダに作成。
Copy All UrlsでブラウザのタブのURL一覧を取得したら、Windowsボタン→qcqc として実行。
後は指示に従えばYoutube動画のURLだけがクリップボードにコピーされますので、
テキストファイルにでも貼り付けて置いてください。
